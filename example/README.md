# Example Implementation of Configon

## Setting up

```
    npm install gulp -g
```

```
    npm install
```

That's all you're ready.

## Try the following

```
    gulp config --env:dev
    gulp config --env:device
    gulp config --env:device --ip:192.168.2.11
```

## Or set NODE_ENV environment variable

 ```
    ( SET | export ) NODE_ENV=stage
    gulp config
```
